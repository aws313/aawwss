# تشغيل المشروع محلياً

ملفان أضفتُهما لتشغيل المشروع بنقرة واحدة على ويندوز:

- [run-local.ps1](run-local.ps1): سكربت PowerShell يقوم بالتحقّق من وجود `node` و`pnpm`، يثبت الحزم ثم يفتح نافذة جديدة ويشغّل `pnpm dev` ويفتح المتصفح على `http://localhost:3000`.
- [start-dev.bat](start-dev.bat): ملف باتش يمكن النقر عليه مرتين ليؤدّي نفس الوظيفة على Windows.

تعليمات سريعة:

1. افتح مستكشف الملفات واذهب إلى مجلد المشروع (حيث يوجد `package.json`).
2. أنقر مزدوجًا على `start-dev.bat` أو افتح PowerShell ومن ثم نفّذ:

```powershell
.\run-local.ps1
```

ملاحظات مهمة:
- لا أستطيع تشغيل الخادم على جهازك عن بُعد؛ هذه السكربتات تُبسّط التشغيل محليًا.
- إن أردت رابطاً عاماً لعرض المشروع (مشاركة عبر الإنترنت)، أقدر أعدّ ملف نشر على Vercel أو Netlify؛ ستحتاج لتفويض قصير أو حساب لنشره.

في حال ظهرت أخطاء، قم بتشغيل `pnpm dev` من نافذة الأوامر وأرسل لي المخرجات أو الملف `dev.log` وسأوجّهك للحل.
